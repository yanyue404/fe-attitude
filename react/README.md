# README

## 什么是 React

- Pwa: 单页面应用
- Base：jsx+webpack+babel+es6
- 专注 UI：用于构建用户界面的 JavaScript 库，声明式编写
- 组件化, state 与 props 构成的单向数据流
- 虚拟 DOM，是轻量的 js 对象，只保留了原生 dom 的一些常用的属性和方法
